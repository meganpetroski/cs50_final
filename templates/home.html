Heartbeat How-To:


This website is designed to display the results of the Heartbeat query system. This sysytem runs nightly on our boundary data products.
Users can view results with a product name and run number on the Results page. The Query page shows the queries and gives a brief description of each used to obtain the result. Finally, the Comparison page allows a user to pick 2 runs for a product and download the CSV representation and upload them to compare them against one another. Below you can view the external release calendar, which tells you when each product is due for 2018.
